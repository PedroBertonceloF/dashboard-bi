# TrabalhoTopicosX — WEB-02 BI Dashboard

Estudo comparativo de geração de código assistida por IA, feito para a disciplina de Tópicos Especiais em
Computação. O mesmo problema (**WEB-02 — Dashboard de BI/Analytics a partir de CSV**) foi resolvido de forma
independente por duas ferramentas:

- [`Gemini/`](./Gemini/) — implementação gerada com o **Gemini CLI** (agente autônomo de terminal)
- [`Claude/`](./Claude/) — implementação gerada com o **Claude** (assistente de chat)

## 📊 Apresentação comparativa

👉 Abra [`apresentacao.html`](./apresentacao.html) no navegador para ver o problema, a metodologia, os
laboratórios de avaliação, os resultados, o histórico de prompts e a conclusão do estudo — incluindo os dois
dashboards rodando lado a lado.

Cada projeto tem seu próprio backend (FastAPI) e frontend (React + Vite); consulte a documentação dentro de
cada pasta (`Gemini/README.md`, `Claude/QUICK_START.md`) para instruções de execução local.
