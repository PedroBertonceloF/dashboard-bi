# Glossary

*   **Dataset:** The tabular data uploaded by a user via a CSV file. A Dataset belongs to the User who uploaded it.
*   **KPI (Key Performance Indicator):** A top-level aggregated metric derived from a Dataset's Value Column (e.g., Total Sum, Average).
*   **Date Column:** A specific column in the Dataset mapped by the user to represent time. Used for the time-series chart and date filters.
*   **Category Column:** A specific column in the Dataset mapped by the user to represent groupings. Used for the bar chart and category filters.
*   **Value Column:** A specific column in the Dataset mapped by the user to represent numerical measurements. Used to calculate KPIs and chart axes.
*   **Dashboard:** The view that renders KPIs and charts based on a Dataset, filtered by user selections.
